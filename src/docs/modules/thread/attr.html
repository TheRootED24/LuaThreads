stacksize ( lto, stacksize)
Get or set a lua thread objects stacksize attribute

Parameters

  • lto: Lua thread object
  • stacksize: integer value representing stacksize in bytes. (optional)

Usage:

local ret, stacksize = thread.attr.stacksize(lto);

if(ret == 0 ) then
  print(stacksize.." bytes")
end

local new_stacksize = 16384;

ret, new_stacksize = thread.attr.stacksize(lto, new_stacksize)

Return values:

Return 2 vaules, the return value of the function call 0 or 1, and the stack size in bytes, if the call succeeds.
